# QGIF Integration Library

This directory contains the QGIF (Quaternion Graphics Interchange Format) integration components for the Sovereign UI.

## Structure

*   **presets/**: 20+ pre-generated QGIF animation files (.json).
*   **components/**: Svelte components for using QGIFs (Loader, Icon, Transition, Background).
*   **types/**: TypeScript type definitions for QGIF data structures.
*   **stores/**: Svelte stores for managing QGIF state.

## Usage

### Loader

```svelte
<script>
  import QGIFLoader from '$lib/asyl/qgif/components/QGIFLoader.svelte';
</script>

<QGIFLoader size={120} />
```

### Icon

```svelte
<script>
  import QGIFIcon from '$lib/asyl/qgif/components/QGIFIcon.svelte';
</script>

<QGIFIcon icon="success_burst" size={32} />
<QGIFIcon icon="error_shake" size={32} />
```
